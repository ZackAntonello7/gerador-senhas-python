# Gerador de Senhas em Python 🔐

Este é um projeto simples desenvolvido em Python que gera senhas aleatórias seguras.

## 📌 Funcionalidades
- Geração de senha aleatória
- Definição do tamanho da senha pelo usuário
- Uso de letras, números e caracteres especiais

## 🛠 Tecnologias utilizadas
- Python

## ▶ Como executar

1. Instale o Python no computador
2. Baixe o arquivo `gsenhas.py`
3. Execute no terminal:

python gsenhas.py
